# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities privately through GitHub Security Advisories:

https://github.com/charl-kruger/gmode/security/advisories/new

Do not open a public issue for a suspected vulnerability. If you are unsure whether something is security-sensitive, use a private advisory.

## Supported Versions

GMode is pre-1.0. Only the latest minor line receives security updates.

| Version | Supported |
| --- | --- |
| Latest minor | Yes |
| Older minors | No |
